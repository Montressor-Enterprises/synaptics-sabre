AMBA Slave Interface Signals

Port Name I/O Description

presetn

 I

Peripheral reset

This is the global reset of the controller, including all the registers.

Synchronous To: Asynchronous

Registered: N/A

Power Domain: SINGLE_DOMAIN

Active State: Low

pclk

 I

APB bus clock

The rising edge of pclk is used to time all transfers on the APB bus. The minimum required frequency is 15 MHz.

Synchronous To: None

Registered: N/A

Power Domain: SINGLE_DOMAIN

Active State: N/A

psel

 I

APB select

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: High

paddr[11:0]

 I

APB address bus

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: N/A

penable

 I

APB strobe

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: High

pwrite

 I

APB transfer direction

When High, this signal indicates an APB write access; when Low, a read access.

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: High

pwdata[31:0]

 I

APB write data bus

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: N/A

 O

Synchronous To: pclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: N/A
