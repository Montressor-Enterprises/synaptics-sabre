APB Slave Interface Signals

Port Name I/O Description

pclk

 I

APB clock used in the APB interface to program registers.

Synchronous To: None

Registered: N/A

Power Domain: SINGLE_DOMAIN

Active State: N/A

presetn

 I

APB clock-domain reset. Asynchronous assertion, synchronous de-assertion. The reset must be synchronously de-asserted after the rising edge of pclk. DW_apb_uart does not contain logic to perform this synchronization, so it must be provided externally.

Synchronous To: Asynchronous

Registered: N/A

Power Domain: SINGLE_DOMAIN

Active State: Low

penable

 I

APB enable control used for timing read/write operations.

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: High

pwrite

 I

APB Write control.

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: High

pwdata[31:0]

 I

APB write data bus.

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: N/A

paddr[7:0]

 I

APB address bus. Uses the lower bits of the APB address bus for register decode.

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: N/A

psel

 I

APB peripheral select.

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: High

prdata[31:0]

 O

APB read data bus.

Synchronous To: pclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: N/A

pready

 O

The ready signal, used to extend the APB transfer and it is also used to indicate the end of a transaction when there is a high in the access phase of a transaction.

Synchronous To: pclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: High

pslverr

 O

APB3 slave error response signal. The signal issues an error when some error condition occurs, as specified in Slave error response section.

Synchronous To: pclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: High

DMA Interface Signals

Port Name I/O Description

dma_tx_ack

 I

DMA Transmit Acknowledge (Active High) indicates that the DMA Controller has transmitted the block of data to the DW_apb_uart for transmission.

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: High

dma_rx_ack

 I

DMA Receive Acknowledge (Active High) indicates that the DMA Controller has transmitted the block of data from the DW_apb_uart.

Synchronous To: pclk

Registered: No

Power Domain: SINGLE_DOMAIN

Active State: High

dma_tx_req

 O

Transmit Buffer Ready (Active High) indicates that the Transmit buffer requires service from the DMA controller.

Synchronous To: pclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: High

dma_tx_single

 O

DMA Transmit FIFO Single (Active High) informs the DMA Controller that there is at least one free entry in the Transmit buffer/FIFO. This output does not request a DMA transfer.

Synchronous To: pclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: High

dma_rx_req

 O

Receive Buffer Ready (Active High) indicates that the Receive buffer requires service from the DMA controller.

Synchronous To: pclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: High

dma_rx_single

 O

DMA Receive FIFO Single (Active High) informs the DMA controller that there is at least one free entry in the Receive buffer/FIFO. This output does not request a DMA transfer.

Synchronous To: pclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: High
