Parameters for DWC_mipi_csi2_host - 1.54a

These tables define all of the user configuration options for this component. Entries which are grayed-out are disabled (read-only) in the current configuration.

  • Basic Configuration
  • Memory Configuration

Basic Configuration Parameters

Basic Configuration Parameters

Label Value Description
Physical Layer Configuration
Physical Layer D-PHY

Physical Interface. When Combo PHY is selected, the D-PHY has 4 lanes and C-PHY 3 lanes.

License Dependencies:

To generate a DWC_mipi_csi2_host with support for Combo C/D-PHY, you must have one of the associated licenses:

  • DWC_mipi_csi-2_host_combo
  • DWC_mipi_csi-2_hp_host_combo

Values:

  • D-PHY (1)
  • Combo (2)

DefaultValue: D-PHY

Enabled: Always

Parameter Name: CSI2_HOST_PHY
Synopsys D-PHY Automatic Integration 1

Enables Synopsys D-PHY as an internal sub-module in the design.

You need to have access to the Synopsys D-PHY package and setup the following variables:

  • setenv CSI2_HOST_PHY_PATH <path where D-PHY is installed>
  • setenv CSI2_HOST_PHY_LIBNAME <path to the .lib or .db file>

Furthermore, you may also need to set the variables for the PLL:

  • setenv CSI2_HOST_PLL_PATH <path where D-PHY PLL is installed>
  • setenv CSI2_HOST_PLL_LIBNAME <path to the pllname.db file>

Values: 0, 1

DefaultValue: 0

Enabled: CSI2_HOST_PHY==1

Parameter Name: CSI2_HOST_SNPS_PHY
Number of PHY Test Control Interfaces 1

Selects the number of PHY Test Control Interfaces.

Values: 0, 1, 2

DefaultValue: =[<functionof>]

Enabled: CSI2_HOST_CPHY==0 && CSI2_HOST_SNPS_PHY==0

Parameter Name: CSI2_HOST_NR_PHY_TESTCTRL_IF
Maximum Number of D-PHY Data Lanes 2

Selects the maximum number of data lanes supported by the D-PHY.

Values: 1, 2, 3, 4, 5, 6, 7, 8

DefaultValue: 4

Enabled: CSI2_HOST_PHY==1

Parameter Name: CSI2_HOST_DPHY_NUMBER_OF_LANES
PHY Protocol Interface (PPI) Data-Processing PIPELINE

Select the type of data processing on the PPI interface:

  • PIPELINE: Pipeline stage on the PPI interface
  • DESKEW: Deskew stage on the PPI interface

Values:

  • PIPELINE (1)
  • DESKEW (2)

DefaultValue: PIPELINE

Enabled: ((CSI2_HOST_PHY==1) && (CSI2_HOST_SNPS_PHY==0)) ? 1 : 0

Parameter Name: CSI2_HOST_DATA_PROCESSING
Number of Pipeline Stages 3

Includes a pipeline of variable depth on the PPI interface between D-PHY and DWC_mipi_csi2_host. This provides flexibility in the placement of D-PHY with respect to DWC_mipi_csi2_host.

This increases the latency through the system. Therefore, the overall control of the system parameters must work effectively so that the data gets through the controller to the application.

Values:

  • 1 (1)
  • 2 (2)
  • 3 (3)
  • 4 (4)

DefaultValue: 1

Enabled: CSI2_HOST_DATA_PROCESSING == 1

Parameter Name: CSI2_HOST_PPI_PD
Generic Configuration
System Interface(s) IPI

System interface(s):

  • IDI: Image Data Interface
  • IPI: Image Pixel Interface
  • IDI&IPI: Both IDI and IPI interfaces

When IPI interface is active, it increases the latency through the system. Therefore, the overall control of the system parameters must work effectively so that the data gets through the controller to the application.

Values:

  • IDI (1)
  • IPI (2)
  • IDI&IPI (3)

DefaultValue: IDI&IPI

Enabled: Always

Parameter Name: CSI2_HOST_DATAINTERFACE
Number of Data IDs to Monitor 8

Selects the number of Data IDs (virtual channel / data type) for which the design is able to detect errors related to line boundary packets.

Values:

  • None (0)
  • 4 (1)
  • 8 (2)

DefaultValue: None

Enabled: Always

Parameter Name: CSI2_HOST_N_DATA_IDS
Free Running APB Clock 0

Specifies if pclk is a free running or a gated clock. If not set, an additional input pin is created.

This pin needs to receive a free running version of pclk.

Values: 0, 1

DefaultValue: 0

Enabled: Always

Parameter Name: CSI2_HOST_PCLK_FREE
Number of Synchronization Stages 3

Selects the number of synchronization stages used for clock domain crossing.

All stages capture the data on the rising edge of the clock.

Values:

  • 2 (2)
  • 3 (3)
  • 4 (4)

DefaultValue: 2

Enabled: Always

Parameter Name: CSI2_HOST_DFLT_F_SYNC_TYPE
Internal PPI Pattern Generator 1

Enables PPI Pattern Generator.

This PPI level pattern generator enables Controller's functional behavior to be independently tested, and possible PHY and/or PHY-CTRL interface errors' cause(s) to be correctly identified. Mandatory for Automotive Package Support.

Values: 0, 1

DefaultValue: 0

Enabled: (CSI2_HOST_AP==0) ? 1 : 0

Parameter Name: CSI2_HOST_PPI_PG
Image Pixel Interface (IPI)
Advanced Pixel Interface Support 1

Enable Advanced Pixel Interface. Extra useful pins is included on IPI.

Values: 0, 1

DefaultValue: 0

Enabled: ((CSI2_HOST_DATAINTERFACE == 2) || (CSI2_HOST_DATAINTERFACE == 3))

Parameter Name: CSI2_HOST_ADVANCED_IPI_IF

Values: 1, 2, 3, 4

DefaultValue: 1

Enabled: ((CSI2_HOST_IPI_IF) && (CSI2_HOST_CSI_HP==0))

Parameter Name: CSI2_HOST_NR_IPI_NHP
Image Data Interface (IDI)
64-bit Mode 1

Values: 0, 1

DefaultValue: 1

Enabled: ((CSI2_HOST_RCV_40B==0) && (CSI2_HOST_IPI_IF == 0) && (CSI2_HOST_CPHY==0) && (CSI2_HOST_AP==0))

Parameter Name: CSI2_HOST_IDI_64_DATA_IF
High-Performance features
CSI-2 High-Performance Features 0

Enables High-Performance features.

License Dependencies:

To enable CSI-2 High-Performance features,you must have one of the associated licenses:

  • DWC_mipi_csi-2_hp_host
  • DWC_mipi_csi-2_hp_host_combo

Values: 0, 1

DefaultValue: 0

Enabled: Always

Parameter Name: CSI2_HOST_CSI_HP
Number of IPIs - up to 8 1

Select the Number of IPI interfaces in the controller

Dependencies:

CSI-2 High-Performance features

Values: 1, 2, 3, 4, 5, 6, 7, 8

DefaultValue: 1

Enabled: ((CSI2_HOST_IPI_IF) && (CSI2_HOST_CSI_HP==1))

Parameter Name: CSI2_HOST_NR_IPI_HP
D-PHY PPI-16 Support 0

Enables support for 16 bit transmission using D-PHY. When PPI 16 is selected the D-PHY will operate using up to 4 lanes

Dependencies:

CSI-2 High-Performance features

Values: 0, 1

DefaultValue: 0

Enabled: ((CSI2_HOST_PHY==1) | (CSI2_HOST_PHY==2)) && (CSI2_HOST_DPHY_NUMBER_OF_LANES<=4) && (CSI2_HOST_CSI_HP==1) && (CSI2_HOST_SNPS_PHY==0)

Parameter Name: CSI2_HOST_DPHY_16
