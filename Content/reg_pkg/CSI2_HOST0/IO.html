AMBA APB Device Interface Signals

Port Name I/O Description

presetn

 I

Peripheral Reset.

Global reset of the controller including all registers. The de-assertion of the signal must be synchronous to pclk.

Synchronous To: None

Registered: N/A

Power Domain: SINGLE_DOMAIN

Active State: Low

pclk

 I

APB Bus Clock.

The rising edge of pclk is used to time all transfers in the APB bus. The minimum required frequency is 15 MHz.

Synchronous To: None

Registered: N/A

Power Domain: SINGLE_DOMAIN

Active State: N/A

fpclk

 I

Free Running Version of APB Bus Clock pclk (Optional).

In case the clock signal pclk is not always available during normal operation (for example, gating of pclk when not using the APB interface, in order to save power consumption), then a free running version of pclk clock is required. If pclk is a free running clock, then this input is not required and should be removed using the associated coreConsultant option.

Synchronous To: None

Registered: N/A

Power Domain: SINGLE_DOMAIN

Active State: N/A

ipi_end_line

 O

End line indication for IPI. Function of this pin is available when:

  • Image frame includes line synchronization packets, both Line Start (LS) and Line end (LE);
  • Frame Start do not trigger any sync event (IPI_ADV_FEATURES[24] = 0);
  • Controller operating in camera timing (IPI_MODE[0] = 0).

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: High

ipi_data_valid[2:0]

 O

Number of valid pixel(s)/byte(s) per clock cycle that are being transmitted on IPI.

  • 000: 1 Valid Pixel Component
  • 001: 2 Valid Pixel Component
  • 010: 3 Valid Pixel Component
  • 011: 4 Valid Pixel Component
  • 100: 5 Valid Pixel Component
  • 101: 6 Valid Pixel Component
  • 110: 7 Reserved
  • 111: 8 Reserved

These values are valid only when ipi_pixen or ipi_embedded are set to 1.

Synchronous To: pixclk

Registered: Yes

Power Domain: SINGLE_DOMAIN

Active State: N/A
