[4:2]

ratio

0x0

used to define the image downsizing ratio in subsampling/binning operation.

1. imgResOpr = 2'b01 (subsampling)

1.1 GrayScale image, ratio is used to specify how many pixels will be skipped to pick up next valid pixel in each H/V direction, please note: the first pixel after cropping is always treated as the first valid output pixel

1.2 Bayer image, ratio is used to specify how many 2x2 quad-pixel will be skipped to pick up next valid quad-pixel in each H/V direction, please note: the first quad-pixel after cropping is always treated as the fisst valit output quad-pixel.

the output picture size after subsampling: 1/(ratio+1)^2, image is in original size if ratio is set to 0 (same effect as imgResOpr = 2'b00)

2. imgResOpr = 2'b10 (binning)

2.1 GrayScale image, ratio is used to specify how many pixels (ratio+1) will be averaged to calculate one output pixel in each H/V direction.

2.2 Bayer image, ratio is used to specify how many quad-pixel (ratio+1) will be averaged to calculate one output quad-pixel in each H/V direction.

the output picture size after binning is: 1/(ratio+1)^2, image is in original size if ratio is set to 0 (same effect as imgResOpr = 2'b00)
