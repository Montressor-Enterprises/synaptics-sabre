Parameters for DWC_mipicsi2_device - 1.32a

These tables define all of the user configuration options for this component. Entries which are grayed-out are disabled (read-only) in the current configuration.

Physical Layer Configuration
Use Synopsys D-PHY Automatic Integration 1

This parameter enables the Synopsys D-PHY as an internal sub-module in the design.

You need to have access to the Synopsys D-PHY package and set up the following variables:

  • setenv CSI2_DEVICE_PHY_PATH <path where D-PHY is installed>
  • setenv CSI2_DEVICE_PLL_PATH <path where D-PHY is installed>
  • setenv CSI2_DEVICE_PHY_LIBNAME <.lib file name>
  • setenv CSI2_DEVICE_PLL_LIBNAME <.lib file name>

Values: 0, 1

DefaultValue: 0

Enabled: Always

Data Interface FIFO Configuration / IDI FIFO Parameters

Data Interface FIFO Configuration / IDI FIFO Parameters

Label Value Description
IDI Header FIFO Configuration
IDI Header FIFO Depth 8

This parameter specifies the depth of the IDI header FIFO. This FIFO is implemented with logic gates (flip-flops).

Values: 4, ..., 256

DefaultValue: 8

Enabled: CSI2_DEVICE_IDI_IF == 1

Parameter Name: CSI2_DEVICE_IDI_HD_FIFO_DEPTH
IDI Payload FIFO Width
External Payload RAM Width for IDI 64

The width of the external payload RAM for IDI

Values: -2147483648, ..., 2147483647

DefaultValue: 64

Enabled: 0

Parameter Name: CSI2_DEVICE_IDI_PLD_WIDTH
IDI Payload FIFO Configuration with DPRAM
IDI Payload FIFO Depth with DPRAM 1024

This parameter specifies the depth of the IDI payload FIFO. This FIFO requires an external asynchronous DPRAM.

Values: 128, ..., 16384

DefaultValue: 1024

Enabled: ((CSI2_DEVICE_USE_DPRAM == 1) && (CSI2_DEVICE_IDI_IF == 1))

Parameter Name: CSI2_DEVICE_IDI_PLD_FIFO_DEPTH
Required IDI Payload DPRAM Depth 1024

This parameter specifies the depth of the RAM required for the IDI payload FIFO.

  • If value of CSI2_DEVICE_IDI_PLD_FIFO_DEPTH parameter is odd, the value of CSI2_DEVICE_IDI_PLD_RAM_DEPTH is required to be (CSI2_DEVICE_IDI_PLD_FIFO_DEPTH + 1).
  • If value of CSI2_DEVICE_IDI_PLD_FIFO_DEPTH parameter is even but not an integer power of two, the value of CSI2_DEVICE_IDI_PLD_RAM_DEPTH is required to be (CSI2_DEVICE_IDI_PLD_FIFO_DEPTH + 2).

Values: -2147483648, ..., 2147483647

DefaultValue: 1024

Enabled: 0

Parameter Name: CSI2_DEVICE_IDI_PLD_RAM_DEPTH
IDI Payload FIFO Configuration with SPRAM
IDI Payload FIFO Depth with SPRAM 1024

This parameter specifies the total depth of the IDI payload FIFO. This FIFO requires two external SPRAM. Provided value must be a multiple of 2.

The total depth includes:

  • External RAM depth.
  • Internal CDC FIFO depth which is based on the level of almost full and cdc sync stage.

Values: 128, ..., 16384

DefaultValue: 1024

Enabled: ((CSI2_DEVICE_USE_SPRAM == 1) && (CSI2_DEVICE_IDI_IF == 1))

Parameter Name: CSI2_DEVICE_IDI_PLD_FIFO_TOTAL_DEPTH
Required IDI Payload PTP SPRAM Depth 1010

This parameter specifies the depth of RAM to the underlying PTP RAM controller of the IDI payload FIFO. This depth should be equally divided by the external odd and even ram.

Values: -2147483648, ..., 2147483647

DefaultValue: 1010

Enabled: 0

Parameter Name: CSI2_DEVICE_IDI_PLD_PTP_SPRAM_DEPTH
