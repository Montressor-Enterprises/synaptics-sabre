DNOTITRANS

0x0

ARM recommendation is based on single layer AHB bus with a

simple fixed priority arbiter.

In our case we have two differences:

1. Our arbiter is not fixed but a round robin

a. Priority will be alternating so effect is less harmful in

case of contention

2. We use a multilayer AHB

a. In many cases the M4 will access data from one memory (e.g.

LP_SENSE memory) while reading instructions form LP_PROC memory

block without contentions.

b. If no contention exist then using DNOTITRANS==1 will degrade

performance
